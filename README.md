# CPA Advisory Starter

This is a beginner-friendly static website project for learning HTML, CSS, JavaScript, Git, and GitHub with a CPA-focused example.

## How to Open

Open `index.html` in a browser.

## What Each File Does

- `index.html` holds the page content.
- `style.css` controls the visual design.
- `script.js` adds the form interaction.

## Daily Git Workflow

```powershell
cd D:\Desktop\my-project
git status
git add .
git commit -m "Describe what changed"
git push
```
